# Pretraining dataset

We curated a large-scale energy consumption dataset consisting of 1.26 billion hourly power consumption observations collected from 76,217 real-world buildings, encompassing both commercial and residential types across diverse countries and temporal spans. The dataset can be downloaded from [here](https://drive.google.com/drive/folders/1jW7QJJv9U8cY5GFU6OIqF49e9g4X-ovp?usp=sharing)
